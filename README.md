# Osna Plotter Website

## Lokal starten
1. Node.js 18 oder 20 installieren
2. Abhaengigkeiten installieren:
   npm install
3. Start:
   npm run dev

## Deploy
- GitHub push
- Vercel import
- Deploy klicken

## Assets
- public/logo.png: Logo
- public/hero.mp4: Hero Video
- public/before.jpg und public/after.jpg: Vorher Nachher Section
- public/work-*.png: Portfolio
